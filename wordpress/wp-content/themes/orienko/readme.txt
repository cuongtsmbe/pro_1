NOTE: How to update theme?
--> Remove the last main theme and upload new main theme in package downloaded. Please make sure you did not customize any thing in main theme last time.
If you customized something, please backup first before upload new theme version then contact to our support to solved issues.
Please don't overwrite theme folder, remove last folder then upload new.

How to update purchased plugins (Visual composer, theme helper, Revolution Slider, Main mega menu)?
---> Delete that plugin then appear missing plugin message, click to install again from theme package.

If you use translate languages please backup languages folder first then overwrite after upload new theme version.
